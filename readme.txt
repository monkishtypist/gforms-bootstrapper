Add Bootstrap CSS classes and styles to Gravity Forms.

== Description ==

Gforms Bootstrapper is a Gravity Forms add-on for adding Twitter Bootstrap classes and CSS styles to forms.

This plugin requires Gravity Forms plugin and Bootstrap CSS to be installed.

== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Updated styles will automatically be applied to forms

== Frequently Asked Questions ==

= Can I use a customized version of Bootstrap? =

Yes. Include your customized Bootstrap SASS files in `/src/sass/app-public.scss` and recompile by running `$ gulp styles`.
